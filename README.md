* **疫情统计数据**  
使用爬虫获取数据并完成前后端交互

* **日期选择**  
<img src="https://img2020.cnblogs.com/blog/1929986/202003/1929986-20200316152034855-1711527543.png" width="600px" height="350px" />

* **全国疫情地图**，颜色的深浅表示疫情的**严重程度**，可以直观了解**高危区域**。
<img src="https://img2020.cnblogs.com/blog/1929986/202003/1929986-20200316153208569-246418803.png" width="600px" height="350px" />

* 鼠标移到每个省份会**高亮显示**和**显示具体疫情情况**。
![](https://img2020.cnblogs.com/blog/1929986/202003/1929986-20200316153341443-1995929822.png)

* 点击某个省份显示该省最近十日的**疫情趋势折线图**，能按日期查看截止当日该省的**确诊、治愈、死亡人数**。 
<img src="https://img2020.cnblogs.com/blog/1929986/202003/1929986-20200316152055758-871940865.png" width="600px" height="350px" />

* 全国最近一周的**疫情趋势折线图**，能按日期查看截止当日全国的**确诊、疑似、治愈、死亡人数**。
<img src="https://img2020.cnblogs.com/blog/1929986/202003/1929986-20200316152108781-2012482868.png" width="600px" height="350px" />
* 可以选择**显示或隐藏**其中某条折线。
<img src="https://img2020.cnblogs.com/blog/1929986/202003/1929986-20200316152631023-401475501.png" width="600px" height="350px" />
